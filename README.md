# Sales Management System

A web-based sales management system built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - Main index page
- `pages/` - Individual HTML pages for different functionalities
- `assets/`
  - `css/` - Stylesheets
  - `js/` - JavaScript files
  - `images/` - Image assets

## Features

- Dashboard
- Customer management
- Product management
- Order management
- Sales records
- Purchase management
- Supplier management
- Feedback system

## Getting Started

Open `index.html` in a web browser to start.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
